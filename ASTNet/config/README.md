## Configuration Files

- **ped2_wresnet.yaml**: training only on the Ped2 dataset and testing on Ped2  
- **ped2_synth_wresnet.yaml**: training with 50% real data and 50% synthetic data (manual selection)  
- **ped2_synth_2_wresnet.yaml**: training with 75% real data and 25% synthetic data (manual selection)  
- **ped2_synth_3_wresnet.yaml**: training with 90% real data and 10% synthetic data (manual + random selection)  
- **ped2_synth_random_wresnet.yaml**: training with 50% real data and 50% synthetic data (random selection)  
- **ped2_synth_2_random_wresnet.yaml**: training with 75% real data and 25% synthetic data (random selection)  
- **synth_wresnet.yaml**: training using only synthetic data (114 videos) and testing on 56 videos (28 normal, 28 anomalous)  
- **med_wresnet.yaml**: testing on the MED dataset (31 videos, ~45,000 frames)
